# Gleam Snippets for Zed IDE

A collection of Gleam snippets for the [Zed IDE](https://zed.dev) that you might be used to from other editors.

## Features

This extension provides a comprehensive set of snippets for Gleam development, including:

- Package and import declarations
- `case` and `use` expressions
- `let` and `const` assignment expressions
- Public, private, test and anonymous function declarations
- Type declarations
- Pipes
- And much more!

## Installation

### Method 1

1. Go to Extensions menu in Zed IDE
2. Search for "gleam-snippets"
3. Click "Install"

### Method 2

1. Clone this repo:

```
git clone https://github.com/BlueBlizzardd/gleam-zed-snippets
```

2. Go to Extensions menu in Zed IDE
3. Click "Install Dev Extension"
4. Select the folder you cloned

## Usage

Start typing the snippet prefix in a Gleam file and press `Tab` to expand the snippet.

## Available Snippets

| Prefix | Description |
|--------|-------------|
| `pub` / `pub fn` | Public function |
| `fn` | Private function |
| `test` | Test function |
| `af` | Anonymous function |
| `let` | Let assignment |
| `leta` | Let assert assignment |
| `const` | Const assignment |
| `pub` / `pub const` | Public const assignment |
| `use` | `use` expression |
| `case` | `case` expression |
| `type` | Type declaration |
| `pty` / `pub type` | Public type declaration |
| `pexfn` / `pub external` / `pub external fn` | Public external function declaration |
| `im` | Import |
| `im.` | Import unqualified |
| `p` / `\|>` | Pipe |
| `#` | Tuple |
| `bl` | Block |
| `seq` / `should` | Should equal |
| `strue` / `should` / `should.b` | Should be true |
| `sfalse` / `should` / `should.b` | Should be true |
